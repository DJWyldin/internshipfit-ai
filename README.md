# InternshipFit AI

AI-powered internship match tool for college students.

Upload your resume and paste any internship description. Get a match score, strengths, weaknesses, missing keywords, tailored cover letter, interview prep questions, and an action plan — in seconds.

## Live App
https://internshipfit-ai.vercel.app

## Built With
- Next.js
- TypeScript
- Tailwind CSS
- OpenAI API
